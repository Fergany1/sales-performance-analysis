# Sales Performance Data Analysis

## Project Overview
This project analyzes sales data to extract insights about revenue performance, top products, and sales trends.

## Objectives
- Analyze total revenue
- Identify top-performing products and categories
- Analyze monthly sales trends
- Evaluate country-level performance

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Smartphones generated the highest revenue
- January had higher sales than February
- Egypt was the top-performing country
